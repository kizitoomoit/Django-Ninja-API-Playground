# Django-Ninja-API-Playground
Just playing around with Django Ninja API, I love the framework's simplicity.

# To run this project:
1. Make sure you have python (preferably 3.14+) and pip installed
2. Clone this repository
3. Cd to the project directory
3. Activate your virtual environment 
4. run ``python -m pip install -r requirements.txt``
4. Migrate database using ``python manage.py makemigrations && python manage.py migrate``
5. Run ``python manage.py runserver``
6. Visit ``127.0.0.1:8000:/api/docs`` for OpenAPI
7. Have fun!
